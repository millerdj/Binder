## binder

#### A mobile web app for people to search for new beers.

### Features
* View many beers
* Favorite a beer
* Edit your search parameters


#### Technologies Used
* React
* Redux.js
* JSX
* Node.js
* Express
* BreweryDB api
* Bootstrap 4.0.0
* Webpack
* Git
